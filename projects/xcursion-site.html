Overview

The challenge: build a site that feels as considered as the brand it represents, without a framework, without a template, without a CMS. Just HTML, CSS, and JavaScript — written by hand, understood completely.

The result is a full-screen sectioned scroll experience with a WebGL-inspired animated grain overlay, custom cursor with lag interpolation, blob-animated backgrounds, and a veil transition system between sections. Every interaction is intentional. Nothing is default.

Tools & Stack

Zero dependencies. All effects are pure CSS animations and vanilla JS. The grain uses an SVG feTurbulence filter animated with CSS keyframes. The background blobs use CSS custom properties and keyframe animations. The cursor uses requestAnimationFrame for smooth lag interpolation.
